OUTCOME
New Visual Language & Core Orchestration UI
Problem

The original AI Studio was built for speed of delivery, resulting in a fragmented user experience that struggled to keep pace with the rapidly evolving agentic AI landscape.

Developers found it difficult to visualize complex agent relationships and debug multi-step autonomous workflows, leading to increased development time and user frustration.

Research

We conducted deep-dive interviews with enterprise developers and technical architects to understand their mental models for "chaining" AI tasks.

Key findings revealed that the biggest pain point wasn't writing the code, but understanding the system's state at any given point in an autonomous loop.

Solution

The redesign introduced a node-based visual orchestration layer that allowed developers to drag, drop, and connect agents like Lego blocks.

We created a unified design system specifically for AI operations, emphasizing clarity, real-time feedback, and high-density information display without the clutter.
