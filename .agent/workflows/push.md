This workflow automates the process of saving your work and syncing it with the remote repository on GitHub.

1. Stage all changes
// turbo
2. git add .

3. Commit changes with a descriptive message
// turbo
4. git commit -m "update: automated portfolio update"

5. Push to the main branch
// turbo
6. git push origin main
